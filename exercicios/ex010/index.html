Usando links externos

Link externo é aquele que te levará para outro site. Ex.: repositório público no Github

Você também pode acessar o canal gratuito no YouTube.

ATENÇÃO: sempre que for um link externo, para evitar que o usuário saia do seu site e não retorne mais, use as tags Target e Rel. Farão com que o link externo seja aberto em uma nova guia, mantendo a guia do seu site

Usando links internos

O link interno fará a relação com ontra página que está dentro do nosso servidor. Fazem parte do mesmo site
